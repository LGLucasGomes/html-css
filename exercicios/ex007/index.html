Formação de texto

Negrito:

Forma não Semantica pra negrito, usando o B. Forma Semantica pra negrito, usando o strong.

Enfase:

Forma não semantica pra italico/enfase usando o i. Forma semantica de usar o italico/enfase, usando o em.

Marcadores:

Para marcar o texto usamos a tag Mark.

Textos grandes e pequenos:

Usando as tags big e small.

Texto Deletado:

Usando del pra fazer o texto cmo excluido.

Texto inserido:

Usando a tag ins pra fazer o texo cm adicionado depois Tambem existe o sublinhado cm a tag U.

Sobrescrito:

Para inserir coisas do tipo x20+3. Usando a tag sup.

Subscrito:

Para inserir coisas do tipo H2O.Usando a tag sub.
